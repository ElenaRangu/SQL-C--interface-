File Leggimi di Microsoft SQL Server 2008

27/06/2008 4:43:51 PM

Nel presente file Leggimi sono contenute importanti informazioni da leggere prima dell'installazione di Microsoft® SQL Server® 2008. Viene specificato come individuare indicazioni sull'installazione nella documentazione in linea, come accedere a Preparazione aggiornamento e alle informazioni aggiuntive sul prodotto disponibili nelle note sulla versione e vengono fornite informazioni sul copyright per il prodotto.

Il file Leggimi è disponibile all'interno dei supporti di installazione nel percorso <unità>:\, nonché nella pagina di download del file Leggimi.

Utilizzare la documentazione in linea per indicazioni sull'installazione

La documentazione in linea di SQL Server 2008 è la fonte principale di informazioni sull'installazione di SQL Server. È possibile ottenere la documentazione in linea nei modi seguenti:

  • Installarla mediante l'opzione di installazione della documentazione relativa alla versione corrente di SQL Server 2008. Dopo l'installazione, eseguire l'aggiornamento all'ultima versione disponibile alla pagina di download della documentazione in linea.
  • Visualizzarla in MSDN.
  • Scaricarne una versione autonoma dalla pagina di download della documentazione in linea.

Per visualizzare le informazioni relative all'installazione della documentazione in linea senza accedere a essa, è possibile aprire il file s10ch_setup.chm dai supporti di installazione o dal computer utilizzando i percorsi indicati di seguito.

  • Per aprire il file s10ch_setup.chm dai supporti di installazione, utilizzare il percorso <unità>:\[x86] | [x64] | [ia64]\help\1040\s10ch_setup.chm.
  • Per aprire il file s10ch_setup.chm nel computer, utilizzare il percorso <drive>:\%Programmi%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1040\s10ch_setup.chm.

Di seguito sono riportati gli argomenti della documentazione in linea disponibili in MSDN che contengono importanti informazioni sull'installazione di SQL Server 2008. Questi argomenti sono presenti anche nelle versioni installate della documentazione in linea e nel file s10ch_setup.chm.

  • Installazione iniziale
  • Requisiti hardware e software
  • Procedura: Installazione di SQL Server 2008
  • Procedura: Aggiornamento a SQL Server 2008
  • Procedura: Installazione di SQL Server 2008 dal prompt dei comandi
  • Procedura: Creazione di un nuovo cluster di failover di SQL Server 2008
  • Procedura: Disinstallazione di un'istanza esistente di SQL Server (programma di installazione)

Analizzare l'aggiornamento con Preparazione aggiornamento

Lo strumento Preparazione aggiornamento analizza l'istanza corrente di SQL Server e fornisce importanti informazioni di cui è necessario essere a conoscenza prima dell'aggiornamento. Per informazioni su Preparazione aggiornamento di SQL Server 2008, accedere al sito Web Upgrade Advisor.

Leggere le note sulla versione per informazioni aggiornate sul prodotto

Nelle note sulla versione sono disponibili informazioni aggiuntive sui comportamenti del prodotto validi esclusivamente per la versione corrente o informazioni non presenti nell'attuale versione della documentazione in linea. Il documento Note sulla versione è disponibile nella pagina di download delle note sulla versione.
