Microsoft SQL Server 2008 추가 정보

2008/6/27 4:43:51 PM

이 추가 정보 문서에는 Microsoft® SQL Server® 2008 설치 전에 읽을 중요 정보가 들어 있습니다. 온라인 설명서에서 설치 도움말을 찾는 방법, 업그레이드 관리자를 찾는 방법, 릴리스 정보에서 추가 제품 정보를 찾는 방법 및 제품의 저작권 고지가 나와 있습니다.

이 추가 정보 파일은 <drive>:\의 설치 미디어와 추가 정보 다운로드 페이지에 있습니다.

온라인 설명서에서 설치 도움말 보기

SQL Server 2008 온라인 설명서는 SQL Server 설치 도움말을 제공하는 주요 설명서입니다. 다음과 같은 방법으로 온라인 설명서를 사용할 수 있습니다.

  • 이 릴리스의 SQL Server 2008에 대한 설명서 설치 옵션으로 온라인 설명서를 설치합니다. 설치 후 온라인 설명서 다운로드 페이지에서 구할 수 있는 최신 버전으로 업데이트하십시오.
  • MSDN에서 온라인 설명서를 봅니다.
  • 온라인 설명서 다운로드 페이지에서 독립 실행형 버전을 다운로드합니다.

온라인 설명서에 액세스하지 않고 온라인 설명서 설치 문서를 보려면 설치 미디어나 컴퓨터의 다음 경로에서 s10ch_setup.chm 파일을 엽니다.

  • 설치 미디어에서 s10ch_setup.chm을 열려면 <drive>:\[x86] | [x64] | [ia64]\help\1042\s10ch_setup.chm을 엽니다.
  • 컴퓨터에서 s10ch_setup.chm을 열려면 <drive>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1042\s10ch_setup.chm을 엽니다.

MSDN의 다음 온라인 설명서 항목에는 SQL Server 2008 설치에 대한 중요 정보가 들어 있습니다. 설치된 온라인 설명서 버전과 s10ch_setup.chm 파일에도 해당 항목이 있습니다.

  • 초기 설치(SQL Server 2008)
  • 하드웨어 및 소프트웨어 요구 사항
  • 방법: SQL Server 2008 설치
  • 방법: SQL Server 2008로 업그레이드(설치)
  • 방법: 명령 프롬프트에서 SQL Server 2008 설치
  • 방법: 새 SQL Server 2008 장애 조치(Failover) 클러스터 만들기(설치)
  • 방법: SQL Server의 기존 인스턴스 제거(설치)

업그레이드 관리자로 업그레이드 분석

업그레이드 관리자 도구는 현재 SQL Server 인스턴스를 분석하여 업그레이드 전에 알아야 할 중요 정보를 제공합니다. SQL Server 2008 업그레이드에 대한 자세한 내용은 업그레이드 관리자 웹 사이트를 참조하십시오.

릴리스 정보에서 최신 제품 정보 보기

릴리스 정보에서는 이 릴리스에만 적용되거나 이 릴리스의 온라인 설명서에 문서화되지 않은 제품 동작에 대한 추가 정보를 제공합니다. 릴리스 정보 문서는 릴리스 정보 다운로드 페이지에 있습니다.
