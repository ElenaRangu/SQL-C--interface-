Fichier Lisez-moi de Microsoft SQL Server 2008

27/06/2008 4:43:51 PM

Ce fichier Lisez-moi contient des informations importantes à lire avant d'installer Microsoft® SQL Server® 2008. Il explique comment obtenir de l'aide lors de l'installation dans la documentation en ligne, comment localiser le Conseiller de mise à niveau, comment rechercher des informations supplémentaires sur le produit dans les notes de mise à jour, et décrit les attributions de copyright du produit.

Ce fichier Lisez-moi est disponible sur votre support d'installation sur <lecteur>:\, ainsi qu'à partir de la page de téléchargement du fichier Lisez-moi.

Utiliser la documentation en ligne pour obtenir de l'aide lors de l'installation

La documentation en ligne de SQL Server 2008 est la documentation principale pour obtenir de l'aide lors de l'installation de SQL Server. Vous pouvez vous procurer la documentation en ligne des façons suivantes :

  • Installez la documentation en ligne avec l'option d'installation de la documentation pour cette version de SQL Server 2008. Après l’installation, effectuez la mise à jour vers la dernière version, qui est disponible à partir de la page de téléchargement de la documentation en ligne.
  • Consultez la documentation en ligne sur MSDN.
  • Téléchargez une version autonome à partir de la page de téléchargement de la documentation en ligne.

Pour afficher la documentation sur l'installation de la documentation en ligne sans accéder à la documentation en ligne, vous pouvez ouvrir le fichier s10ch_setup.chm à partir du support d'installation ou de votre ordinateur en utilisant les chemins d'accès suivants :

  • Pour ouvrir s10ch_setup.chm à partir du support d'installation, ouvrez <lecteur>:\[x86] | [x64] | [ia64]\help\1036\s10ch_setup.chm.
  • Pour ouvrir s10ch_setup.chm sur votre ordinateur, ouvrez <lecteur>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1033\s10ch_setup.chm.

Les rubriques suivantes de la documentation en ligne sur MSDN contiennent des informations importantes sur l'installation de SQL Server 2008. Ces rubriques figurent également dans les versions installées de la documentation en ligne et dans le fichier s10ch_setup.chm.

  • Installation initiale
  • Configuration matérielle et logicielle requise
  • Procédure : installer SQL Server 2008
  • Procédure : effectuer une mise à niveau vers SQL Server 2008
  • Procédure : installer SQL Server 2008 à partir de l'invite de commandes
  • Procédure : créer un nouveau cluster de basculement SQL Server 2008
  • Procédure  désinstaller une instance existante de SQL Server (programme d'installation)

Analyser la mise à niveau avec le Conseiller de mise à niveau

L'outil Conseiller de mise à niveau analyse votre instance actuelle de SQL Server et fournit des informations importantes à connaître avant de procéder à la mise à niveau. Pour plus d'informations sur le Conseiller de mise à niveau SQL Server 2008, consultez le site Web du Conseiller de mise à niveau.

Lire les notes de mise à jour pour connaître les dernières informations sur le produit

Les notes de mise à jour proposent des informations supplémentaires sur les comportements du produit qui s'appliquent uniquement à cette version ou qui ne sont pas documentés dans cette version de la documentation en ligne. Le document des notes de mise à jour est disponible à partir de la page de téléchargement des notes de mise à jour.
