Microsoft SQL Server 2008-Infodatei

27/6/2008 4:43:51 p.m.

Diese Infodatei enthält wichtige Informationen, die Sie vor der Installation von Microsoft® SQL Server® 2008 lesen sollten. Hier erfahren Sie, wie Sie Hilfe zur Installation in der Onlinedokumentation finden, wie Sie den Upgrade Advisor finden und wie Sie zusätzliche Produktinformationen in den Versionsanmerkungen finden. Außerdem sind Urheberrechtshinweise für das Produkt enthalten.

Sie finden diese Infodatei auf dem Installationsmedium unter dem Pfad <Laufwerk>:\. Sie ist auch auf der Downloadseite für Infodateien verfügbar.

Verwenden der Onlinedokumentation zur Unterstützung bei der Installation

Die SQL Server 2008-Onlinedokumentation stellt die wesentliche Dokumentation für die Unterstützung bei der Installation von SQL Server 2008 dar. Es gibt folgende Möglichkeiten, um auf die Onlinedokumentation zuzugreifen:

  • Installieren Sie die Onlinedokumentation mit der Installationsoption für Dokumentation für diese Version von SQL Server 2008. Führen Sie nach der Installation ein Update auf die nächste Version durch, die über die Downloadseite für die Onlinedokumentation verfügbar ist.
  • Zeigen Sie die Onlinedokumentation auf MSDN an.
  • Laden Sie eine eigenständige Version von der Downloadseite für Infodateien herunter.

Wenn Sie die Installationsinformationen der Onlinedokumentation anzeigen möchten, ohne auf die Onlinedokumentation zuzugreifen, können Sie die Datei 's10ch_setup.chm' unter Verwendung der folgenden Pfade vom Installationsmedium oder von Ihrem Computer aus öffnen:

  • Um 's10ch_setup.chm' vom Installationsmedium aus zu öffnen, öffnen Sie <Laufwerk>:\[x86] | [x64] | [ia64]\help\1031\s10ch_setup.chm.
  • Um 's10ch_setup.chm' auf dem Computer zu öffnen, öffnen Sie <Laufwerk>:\%Programme%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1031\s10ch_setup.chm.

Die folgenden Themen in der Onlinedokumentation auf MSDN enthalten wichtige Informationen zur Installation von SQL Server 2008. Diese Themen befinden sich auch in der installierten Version der Onlinedokumentation sowie in der Datei 's10ch_setup.chm'.

  • Vorgehensweise: Deinstallieren einer vorhandenen SQL Server-Instanz (Setup)

Analysieren Ihrer Aktualisierung mit dem Upgrade Advisor

Das Upgrade Advisor-Tool analysiert Ihre aktuelle Instanz von SQL Server und liefert wichtige Informationen, die Sie vor der Aktualisierung kennen sollten. Weitere Informationen zum Upgrade Advisor für SQL Server 2008 finden Sie auf der Upgrade Advisor-Website.

Lesen der neuesten Produktinformationen in den Versionsanmerkungen

Die Versionsanmerkungen enthalten weitere Informationen zu Produktverhaltensweisen, die nur für diese Version gelten oder in dieser Version der Onlinedokumentation nicht dokumentiert sind. Das Dokument mit den Versionsanmerkungen ist auf der Downloadseite für Versionsanmerkungen verfügbar.
