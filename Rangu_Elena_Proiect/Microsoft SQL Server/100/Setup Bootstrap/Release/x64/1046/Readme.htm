Leiame do Microsoft SQL Server 2008

27/06/2008 4:43:51 PM

Este documento Leiame contém informações importantes que devem ser lidas antes da instalação do Microsoft® SQL Server® 2008. Ele descreve como encontrar ajuda sobre instalação nos Manuais Online, como localizar o Supervisor de Atualização, como encontrar informações complementares do produto nas Notas de Versão e as atribuições de direitos autorais do produto.

Este arquivo Leiame está disponível na mídia de instalação, em <unidade>:\ e também na página de download do Leiame.

Use os Manuais Online para obter ajuda sobre instalação

Os Manuais Online do SQL Server 2008 constituem a principal documentação de ajuda sobre a instalação do SQL Server. Você pode obter os Manuais Online das seguintes maneiras:

  • Instale os Manuais Online com a opção de instalação de documentação desta versão do SQL Server 2008. Depois de instalar, atualize a versão mais recente, que está disponível na página de download dos Manuais Online.
  • Exiba os Manuais Online disponíveis no MSDN.
  • Baixe uma versão autônoma disponível na página de download dos Manuais Online.

Para exibir a documentação de instalação dos Manuais Online sem acessá-los, abra o arquivo s10ch_setup.chm da mídia de instalação ou do computador usando os seguintes caminhos:

  • Para abrir o arquivo s10ch_setup.chm da mídia de instalação, abra <unidade>:\[x86] | [x64] | [ia64]\help\1033\s10ch_setup.chm.
  • Para abrir o arquivo s10ch_setup.chm no computador, abra <unidade>:\%Arquivos de Programas%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1033\s10ch_setup.chm.

Os tópicos a seguir dos Manuais Online disponíveis no MSDN contêm informações importantes sobre como instalar o SQL Server 2008. Esses tópicos também estão nas versões instaladas dos Manuais Online e no arquivo s10ch_setup.chm.

  • Instalação inicial
  • Requisitos de hardware e de software
  • Como instalar o SQL Server 2008
  • Como atualizar para o SQL Server 2008
  • Como instalar o SQL Server 2008 a partir do prompt de comando
  • Como criar um novo cluster de failover do SQL Server 2008
  • Como desinstalar uma instância existente do SQL Server (Instalação)

Analise sua atualização com o Supervisor de Atualização

A ferramenta Supervisor de Atualização analisa sua instância atual do SQL Server e apresenta informações importantes de que você deve ter conhecimento antes de atualizar. Para obter informações sobre o Supervisor de Atualização do SQL Server 2008, consulte o site do Supervisor de Atualização.

Leia as Notas de Versão para obter as informações mais atualizadas sobre o produto

As Notas de Versão fornecem informações complementares a respeito de comportamentos do produto que se aplicam somente a esta versão ou que não estão documentados nesta versão dos Manuais Online. O documento Notas de Versão está disponível na página de download das Notas de Versão.
