Файл Readme для Microsoft SQL Server 2008

27.06.2008г. 16:43:51

Этот документ Readme содержит важную информацию, которую следует прочитать перед установкой Microsoft® SQL Server® 2008. В документе описаны способы поиска справки по установке в электронной документации, способы поиска помощника по обновлению, способы поиска дополнительных сведений о продукте в заметках о версии. Также в документе содержатся указания на авторские права на продукт.

Этот файл Readme доступен на установочном носителе <диск>:\, а также на странице загрузки файла Readme.

Используйте электронную документацию для получения помощи в установке

Электронная документация по SQL Server 2008 является основным источником справочных сведений по установке SQL Server. Получить доступ к электронной документации можно следующими способами.

  • Установить электронную документацию для данной версии SQL Server 2008, включив параметр установки документации. После установки обновления последней версии, которая доступна на странице загрузки электронной документации.
  • Просмотреть электронную документацию в MSDN.
  • Загрузить изолированную версию на странице загрузки электронной документации.

Чтобы просмотреть документацию по установке, не заходя в электронную документацию, можно открыть файл s10ch_setup.chm с установочного носителя или с вашего компьютера по следующим путям.

  • Чтобы открыть файл s10ch_setup.chm с установочного носителя, откройте файл <диск>:\[x86] | [x64] | [ia64]\help\1049\s10ch_setup.chm.
  • Чтобы открыть файл s10ch_setup.chm с вашего компьютера, откройте файл <диск>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1049\s10ch_setup.chm.

  • Как удалить существующий экземпляр SQL Server (программа установки)

Выполните анализ с использованием помощника по обновлению

Помощник по обновлению анализирует текущий экземпляр SQL Server и предоставляет важные сведения, которые необходимо знать перед обновлением. Сведения о помощнике по обновлению SQL Server 2008 см. на веб-узле Помощник по обновлению.

Прочитайте заметки о версии для получения последних сведений о продукте

Заметки о версии дают дополнительные сведения о поведении продукта, которые применимы только к этой версии или не документированы в этой версии электронной документации. Документ заметок о версии доступен на странице загрузки заметок о версии.
