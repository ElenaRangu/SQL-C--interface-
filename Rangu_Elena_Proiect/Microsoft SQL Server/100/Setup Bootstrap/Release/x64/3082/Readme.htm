Archivo Léame de Microsoft SQL Server 2008

27/6/2008 4:43:51 p.m.

Este documento Léame contiene información relevante que se debe leer antes de instalar Microsoft® SQL Server® 2008. Describe la forma de buscar los siguientes elementos: ayuda para la instalación en los Libros en pantalla, el Asesor de actualizaciones e información complementaria del producto en las Notas de la versión; también describe los derechos de copyright para el producto.

Este archivo Léame está disponible en los medios de instalación, en <unidad>:\, así como en la página de descarga del archivo Léame.

Usar los Libros en pantalla como ayuda para la instalación

Los Libros en pantalla de SQL Server 2008 constituyen la documentación principal para obtener ayuda para la instalación de SQL Server. Se pueden obtener de las tres formas siguientes:

  • Instalando los Libros en pantalla con la opción de instalación de documentación para esta versión de SQL Server 2008. Después de completar la instalación, actualice a la última versión, que está disponible en la página de descarga de los Libros en pantalla.
  • Viendo los Libros en pantalla en MSDN.
  • Descargando una versión independiente desde la página de descarga de los Libros en pantalla.

Para ver la documentación de instalación de los Libros en pantalla sin acceso a los mismos, puede abrir el archivo s10ch_setup.chm desde los medios de instalación o desde su equipo usando las rutas de acceso siguientes:

  • Para abrir el archivo s10ch_setup.chm desde los medios de instalación, abra <unidad>:\[x86] | [x64] | [ia64]\help\1033\s10ch_setup.chm.
  • Para abrir el archivo s10ch_setup.chm desde su equipo, abra <unidad>:\%Archivos de programa%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1033\s10ch_setup.chm.

Los siguientes temas de los Libros en pantalla en MSDN contienen información relevante sobre la instalación de SQL Server 2008. Estos temas también se encuentran en las versiones instaladas de los Libros en pantalla y en el archivo s10ch_setup.chm.

  • Instalación inicial
  • Requisitos de hardware y software
  • Cómo instalar SQL Server 2008
  • Cómo actualizar a SQL Server 2008
  • Cómo instalar SQL Server 2008 desde el símbolo del sistema
  • Cómo crear un nuevo clúster de conmutación por error de SQL Server 2008
  • Cómo desinstalar una instancia existente de SQL Server (programa de instalación)

Analizar una actualización con el Asesor de actualizaciones

La herramienta Asesor de actualizaciones analiza la instancia actual de SQL Server y proporciona información relevante que se debe conocer antes de proceder a la actualización. Para obtener información sobre el Asesor de actualizaciones de SQL Server 2008, vea el sitio web del Asesor de actualizaciones.

Leer las Notas de la versión correspondientes a la información más reciente del producto

Las Notas de la versión proporcionan información complementaria acerca de los comportamientos del producto que solo se aplican a esta versión o que no están documentados en esta versión de los Libros en pantalla. El documento denominado Notas de la versión está disponible en la página de descarga de las Notas de la versión.
