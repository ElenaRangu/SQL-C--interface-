Microsoft SQL Server 2008 讀我檔案

2008/6/27 下午 4:43:51

這份讀我文件包含重要的資訊，在您安裝 Microsoft® SQL Server® 2008 之前，請務必先閱讀這份文件。說明如何查閱《線上叢書》中的安裝協助資訊、如何找到 Upgrade Advisor、如何找到「版本資訊」中的輔助產品資訊，以及產品的著作權歸屬事宜。

您可以在安裝光碟中找到讀我檔案，位置是 <磁碟機>:\，也可以在讀我檔案下載頁面下載這個檔案。

透過線上叢書取得安裝協助

《SQL Server 2008 線上叢書》是一套可以讓您取得 SQL Server 安裝協助的主要文件集。您可以使用下列方法取得《線上叢書》：

  • 透過文件集安裝選項來安裝 SQL Server 2008 的《線上叢書》。安裝後，請從線上叢書下載頁面更新至最新的版本。
  • 造訪 MSDN 網站以閱讀《線上叢書》。
  • 瀏覽線上叢書下載頁面下載獨立的版本。

如果要閱讀《線上叢書》的安裝文件集但不存取《線上叢書》，您可以開啟安裝光碟上的 s10ch_setup.chm 檔案，這個檔案也存放在電腦上，路徑分別是：

  • 安裝光碟上的 s10ch_setup.chm，其路徑為 <磁碟機>:\[x86] | [x64] | [ia64]\help\1028\s10ch_setup.chm。
  • 電腦上的 s10ch_setup.chm，其路徑為 <磁碟機>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1028\s10ch_setup.chm。

透過 Upgrade Advisor 進行升級分析

Upgrade Advisor 這套工具可以分析您現有的 SQL Server 執行個體，並且提供升級之前所必須知道的重要資訊。如需有關 SQL Server 2008 Upgrade Advisor 的詳細資訊，請參閱 Upgrade Advisor 網站。

閱讀版本資訊以取得最新產品資訊

「版本資訊」提供了一些輔助資訊，特別針對目前這個版本的產品行為模式，或是這一版的《線上叢書》沒有提及的產品行為模式，提供了相關說明。您可以造訪版本資訊下載頁面下載版本資訊文件。
