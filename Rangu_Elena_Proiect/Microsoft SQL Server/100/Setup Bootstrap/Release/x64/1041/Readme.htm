Microsoft SQL Server 2008 Readme

2008/6/27 4:43:51 PM

この Readme ドキュメントには、Microsoft® SQL Server® 2008 をインストールする前に読む必要がある、重要な情報が記載されています。オンライン ブックでインストール関連情報を探す方法、アップグレード アドバイザを探す方法、リリース ノートの製品補足情報を探す方法、および製品の著作権帰属について説明されています。

この Readme ファイルは、インストール メディアの <drive>:\ に保存されており、Readme ダウンロード ページからも参照できます。

オンライン ブックのインストール関連情報を利用する

SQL Server 2008 オンライン ブックは、SQL Server のインストールに関する情報の主要なドキュメントです。オンライン ブックは以下の方法で利用できます。

  • SQL Server 2008 のこのリリースのドキュメント インストール オプションを使って、オンライン ブックをインストールします。インストールが完了したら、オンライン ブック ダウンロード ページから最新バージョンをダウンロードして更新してください。
  • オンライン ブックを MSDN で参照します。
  • スタンドアロン バージョンをオンライン ブック ダウンロード ページからダウンロードします。

オンライン ブックにアクセスせずに、オンライン ブックのインストール情報を表示するには、インストール メディアまたはコンピュータの以下のパスにある s10ch_setup.chm ファイルを開きます。

  • インストール メディアの s10ch_setup.chm を表示するには、<drive>:\[x86] | [x64] | [ia64]\help\1041\s10ch_setup.chm を開きます。
  • コンピュータ上の s10ch_setup.chm を表示するには、<drive>:\%Program Files%\Microsoft SQL Server\100\Setup Bootstrap\CTP6\[x86] | [x64] | [ia64]\Help\1041\s10ch_setup.chm を開きます。

アップグレード アドバイザでアップグレードを分析する

アップグレード アドバイザ ツールは、SQL Server の現在のインスタンスを分析し、アップグレードの実行前に知っておく必要がある重要な情報を提供します。SQL Server 2008 アップグレード アドバイザの詳細については、アップグレード アドバイザ Web サイトを参照してください。

リリース ノートで最新の製品情報を読む

リリース ノートは、このリリースのみでの製品の動作に関する補足情報や、このリリースのオンライン ブックに記載されていない情報を提供します。リリース ノートは、リリース ノート ダウンロード ページで入手できます。
